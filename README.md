## Objective

The objective of this project is to develop a secure and user-friendly Online Examination System that enables students to register, log in, manage their profiles, take timed examinations, and view their results through a web-based interface.

The system aims to simplify the examination process while ensuring secure authentication and efficient result management.

---

## Features

* User Registration and Login
* Secure Authentication using Spring Security
* Student Dashboard
* Profile Management
* Timed Online Examination
* Automatic Exam Submission
* Result Calculation and Storage
* Result Viewing
* Responsive User Interface
* MySQL Database Integration

2. Implemented secure authentication using Spring Security.

4. Created a student dashboard for navigation.
5. Implemented profile management features.
6. Developed a timed online examination module.
7. Added automatic result calculation after exam submission.
8. Stored examination results in the MySQL database.
9. Created a result viewing module.

11. Integrated all modules into a complete web application.
12. Uploaded and maintained the project on GitHub.
