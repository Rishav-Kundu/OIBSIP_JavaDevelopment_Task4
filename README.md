# Online Examination System

A web-based Online Examination System built using Java Spring Boot, Thymeleaf, Spring Security, MySQL, and Maven.

## Features

* User Registration and Login
* Spring Security Authentication
* Profile Management
* Timed Online Examination
* Automatic Exam Submission
* Result Calculation
* Result History Viewing
* Responsive User Interface
* MySQL Database Integration

## Technologies Used

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* Thymeleaf
* MySQL
* Maven
* HTML/CSS

## Project Structure

* Authentication & Authorization
* Student Dashboard
* Profile Management
* Examination Module
* Result Module
